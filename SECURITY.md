# Security Policy

## Supported status

BridgeHR AI is an early public MVP and is not approved for production data.
There is currently no supported production release.

## Reporting a vulnerability

Please report potential vulnerabilities privately through GitHub's security
advisory feature. Do not include real employee records, credentials or identity
documents in a report. A minimal synthetic reproduction is preferred.

## Data boundary

The public demo runs locally in the browser with bundled synthetic records. It
does not require an API key, database or external AI service. Future features
that process operational HR information will require access control,
encryption, audit logging, retention rules, vendor assessment and legal review.
