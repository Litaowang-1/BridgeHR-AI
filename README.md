# BridgeHR AI

AI Workforce Identity & Compliance Platform for International Construction Projects.

BridgeHR AI is an AI-powered platform that helps HR and Administration teams manage passports, visas, work permits, residence permits, and workforce compliance for international construction projects.

## Features

- Passport Management
- Visa Management
- Work Permit Management
- Residence Permit Management
- Employee Document Tracking
- Smart Expiration Reminders
- AI Compliance Assistant
- AI Document Analysis
- Multilingual Support (Chinese, English, French)

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- OpenAI API
- PostgreSQL (planned)
- Vercel
